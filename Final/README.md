# 통합 AI 에이전트 서비스 기획서

## 1. 핵심 목표

- 로컬 Ollama와 Flask를 연결해 통합 웹 AI 서비스를 구현한다.
- 학습 도우미, 회의록 요약, 여행/맛집 추천, 이력서 코치 기능을 하나의 앱으로 통합한다.
- 사용자가 브라우저에서 직접 체험할 수 있는 AI 에이전트 서비스형 데모로 완성한다.
- 실행 가능하고 설명 가능한 구조로 정리한다.

## 2. 프로젝트 개요

사용자는 웹 페이지에서 기능을 선택하고 입력 내용을 전달하면, Flask 서버가 Ollama 모델로 프롬프트를 전송하고 결과를 반환한다.  

## 3. 기능 설계

- 기능별로 서로 다른 프롬프트를 설계해 목적에 맞는 응답을 생성한다.
- UI에서 기능을 선택할 수 있도록 구성하고, 예시 입력을 함께 제공한다.
- 응답 생성 중 진행 상태(프로그래스바)를 표시해 데모의 완성도를 높인다.
- 회의록 기능은 요약, 핵심 포인트, 액션 아이템을 구조적으로 정리한다.

## 4. 기능 요구사항

### 4.1 AI 개인 학습 도우미
- 사용자가 학습 내용을 입력하면 핵심 개념, 예시, 퀴즈를 포함한 학습 보조 답변을 제공합니다.

### 4.2 AI 회의록 요약 & 액션 아이템 생성기
- 회의 내용을 입력하면 요약, 핵심 포인트, 실행할 액션 아이템을 정리해 보여줍니다.

### 4.3 AI 여행/맛집 추천 서비스
- 취향이나 목적을 입력하면 여행지나 맛집을 추천하고, 이유와 방문 포인트를 함께 제시합니다.

### 4.4 AI 이력서/자소서 코치
- 이력서 문장이나 자소서 문구를 입력하면 문장 개선 방향과 더 강한 표현을 제안합니다.

## 5. 기술 구성

- Python: 서버 로직과 AI 요청 처리
- Flask: 웹 서버 및 API 라우팅
- Ollama: 로컬 LLM 실행 환경
- HTML/CSS/JavaScript: 웹 UI 구현
- SSE(Streaming) 방식: 응답 생성 과정을 실시간으로 표시

## 7. 실행 환경

- OS: Windows 11
- Python: 3.10 이상
- Ollama: llama3.1
- Flask: 3.x

### 7.1 Ollama 설치

Ollama 모델을 설치하거나 실행합니다.

```powershell
ollama run llama3.1
```

현재 기본 모델명은 코드에서 `llama3.1`로 설정되어 있습니다.

### 7.2 Flask 설치

```powershell
pip install flask
```

## 8. 실행 방법

### 8.1 Ollama 실행

```powershell
ollama serve
ollama list
ollama pull llama3.1
```

### 8.2 웹 앱 실행

```powershell
python app_multi.py
```

브라우저에서 아래 주소로 접속합니다.

```text
http://127.0.0.1:5000/
```

## 9. 동작 흐름

1. 사용자가 웹 페이지에서 기능을 선택합니다.
2. 입력한 텍스트가 Flask 서버로 전달됩니다.
3. 서버는 해당 모드에 맞는 프롬프트를 생성합니다.
4. Ollama가 응답을 생성합니다.
5. 생성 결과가 스트리밍 형태로 웹 UI에 표시됩니다.
6. 사용자는 결과를 확인하고 추가 입력을 이어서 테스트할 수 있습니다.
